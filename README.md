# EDA_of_bike-sharing-demand

原始数据为：https://www.kaggle.com/competitions/bike-sharing-demand
清洗后的数据为：cleaned_data.csv

自行车租赁需求受多重因素综合影响，呈现出复杂的动态变化特征：时间维度上，小时、日期、季节、节假日等因素可能导致需求出现周期性波动；环境维度上，温度、湿度、风速、天气类型等气象条件直接影响用户的出行决策；用户维度上，注册用户与临时用户的出行偏好差异可能进一步加剧需求的异质性。这些因素的交互作用，使得需求预测成为一项具有挑战性的任务。基于此，本研究以该数据集为对象，开展探索性数据分析并运用机器学习方法进行预测。通过挖掘时间、气象、用户类型等特征与租赁量的内在关联，揭示自行车租赁需求的变化规律，识别关键影响因素，对需求进行建模预测、优化共享自行车系统的运营策略提供数据支撑与理论参考。
